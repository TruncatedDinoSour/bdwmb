# BdwmB
A simple, modular bash/shell script for configuring your DWM bar.

## Installation
```bash
$ git clone https://github.com/TruncatedDinosour/bdwmb
$ cd bdwmb
$ make full
```

## Flags
```
-q = quiet, do not show debug messages.
```

## Using it in .xinitrc
```bash
exec /usr/local/bin/bdwmb -q &
```

## Dependencies
```
xsetroot
```
